# Student Performance Dataset Analysis and Data Preprocessing using R

This project focuses on analyzing and preprocessing a Student Performance Dataset using R programming. The dataset contains information related to students’ demographic details, study habits, parental involvement, extracurricular activities, GPA, and academic performance categories. The main goal of this project was to clean, transform, and prepare the dataset for accurate statistical analysis and future predictive modeling.

## Key Tasks Performed

* Loaded and explored the dataset using R
* Analyzed dataset structure, dimensions, and summary statistics
* Detected and handled missing values using mean, median, and mode imputation
* Identified and removed duplicate records
* Detected and fixed outliers using the IQR method
* Checked and corrected invalid and noisy data values
* Converted numeric attributes into categorical format
* Filtered student records based on GPA conditions
* Performed feature selection using correlation analysis
* Removed irrelevant attributes such as Height and StudentID
* Calculated descriptive statistics according to GradeClass
* Compared GPA and study habits across Gender categories
* Examined variability using Standard Deviation, Range, and IQR
* Balanced imbalanced classes using Oversampling
* Applied Min-Max Normalization on GPA
* Split the dataset into Training and Testing sets

## Technologies Used

* R Programming
* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Statistical Analysis
* Feature Selection
* Oversampling
* Normalization Techniques

## Dataset

* Student Performance Dataset (Kaggle)

## Project Goal

The primary objective of this project is to improve dataset quality through preprocessing and data cleaning techniques, perform meaningful statistical analysis, and prepare the dataset for future predictive analysis and machine learning tasks.
